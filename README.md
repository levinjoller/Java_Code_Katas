# Java_Code_Katas

![ci master](https://github.com/levinjoller/java_code_katas/workflows/ci/badge.svg)

The tasks are solved by test driven development. This project is for exercise purpose only.

## Build with

- Eclipse v4.19.0
- Apache Maven v3.8.1
- OpenJDK v16.0.1
- JUnit v5

## Usage

Run unit tests:

```sh
mvn test
```
